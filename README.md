# coffee_sales_sqlproject

## Description
This project analyzes sales data from a coffee retailer using SQL. 
It extracts insights about customers, products, and city-wise sales performance.

## Tables Used
- `city`
- `customers`
- `products`
- `sales`

## Queries
1. **Coffee Consumer Count:** Number of coffee drinkers per city
2. **Target Audience:** Potential coffee consumers per city (25% of population)
3. **Total Coffee Sales Last Quarter 2023**
4. **Total Coffee Sales by City Last Quarter 2023**
5. **Units Sold Per Product**
6. **Average Sales per Customer per City**
7. **Top 5 Best-Selling Products**
8. **Top Cities by Number of Sales**

## Usage
1. Import the `sql_customer` database into MySQL Workbench.
2. Run queries from the `queries/` folder.
3. Each query is documented with a header describing its purpose.
